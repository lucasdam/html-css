Natureza dos Links (rel="")

  • next: indica que o link é para a próxima parte do documento atual
  • prev: prev indica que o link é para a parte anterior do documento atual
  • author: indica que é um link para o site do autor do artigo atual
  • external: indica que é um link para outro site que não faz parte do site atual
  • nofollow: indica que é um link para um site não endossado, como um link pago

CSS INTERNO/LOCAL

Adiciona a tag style dentro da tag head. Todos os estilos serão trabalhados dentro desse style. O background-color dessa página inteira foi inserido lá.


CSS EXTERNO

É criado um arquivo novo, do tipo .css, para adicionar todos os estilos.

Para fazer a referência ao arquivo .css, escreva "link rel="stylesheet" href="style.css" entre tags, dentro do head.

Várias páginas html podem usar o mesmo arquivo style.css
